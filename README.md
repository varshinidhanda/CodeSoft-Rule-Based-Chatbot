# Rule-Based Chatbot Using Python

## About the Project

This project is a simple Rule-Based Chatbot developed using Python as part of the CodeSoft Artificial Intelligence Internship.

The chatbot responds to user inputs using predefined rules and keyword matching techniques.

## Features

* Greeting responses
* AI-related responses
* Time response
* Simple conversation flow
* Exit functionality

## Technologies Used

* Python
* VS Code

## Project Working

The chatbot takes user input and checks it using if-else conditions. Based on the detected keywords, it provides suitable responses.

## How to Run the Project

1. Install Python
2. Open the project in VS Code
3. Run the following command:

```bash id="dwl0dc"
python chatbot.py
```

## Sample Inputs

* hi
* what is your name
* tell me about ai
